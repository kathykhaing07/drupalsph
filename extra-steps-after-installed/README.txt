
After installed SPHTechBlog, in order to add crop properties to Images Styles Setting for Large and Medium. 


========== Delete Default Images Styles for Large and Medium ============

1. go to admin/config/media/image-styles/
2. Delete Large and Medium


========== Import Images Styles for Large and Medium ============

1. go to admin/config/development/configuration/single/import
2. Tab to "Single Item"
3. Choose -> Configuration type : Image Style
- 3.1. copy and paste each of image.style.large.yml, image.style.medium.yml and import

P.S. This extra steps caused by changing setting of Image Styles Default
